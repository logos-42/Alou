MCP Host - 智能 MCP 服务管理器
================================

版本: 1.0.0
构建时间: 2025/7/2 20:18:16

使用方法
--------
1. 搜索服务: mcp-host.exe "搜索天气服务"
2. 创建服务: mcp-host.exe "创建一个翻译工具"
3. Web API: mcp-host.exe --server 3000

配置说明
--------
编辑 .env 文件，填入你的 API 密钥。

常见问题
--------
Q: Windows 提示不安全？
A: 右键属性，勾选"解除锁定"。

Q: 运行没有反应？
A: 检查 .env 文件是否在同一目录。
